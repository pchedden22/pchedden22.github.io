Driver Rewards Web App

A web application designed to help trucking companies reward their drivers with a customizable points-based system. Sponsors can allocate points based on performance, and drivers can redeem them for rewards or donate to selected charities. The app features role-based access, real-time points tracking, sponsor-specific settings, and seamless account management. Built with AWS DynamoDB for scalable data storage and secure authentication, the platform ensures a smooth and efficient experience for both sponsors and drivers.
