Projects

Driver Rewards Web App

A web application designed to help trucking companies reward their drivers with a customizable points-based system. Sponsors can allocate points based on performance, and drivers can redeem them for rewards or donate to selected charities. The app features role-based access, real-time points tracking, sponsor-specific settings, and seamless account management. Built with AWS DynamoDB for scalable data storage and secure authentication, the platform ensures a smooth and efficient experience for both sponsors and drivers.

Sepsis Prediction Model

A machine learning model designed for early detection of sepsis using physiological time-series data from the PhysioNet Sepsis Challenge dataset. The model predicts the likelihood of sepsis six hours before clinical recognition by analyzing patient vitals and lab results over time. Implemented custom feature engineering, trained models using logistic regression, SVM, and other techniques, and evaluated performance with cross-validation. The project showcases predictive analytics, medical AI, and time-series data processing.

Connect 4 AI

A digital recreation of the classic Connect 4 game, built with a focus on smooth gameplay and intuitive UI. Players take turns dropping colored discs into a 7x6 grid, aiming to connect four in a row before their opponent. The app features real-time game logic, an interactive interface, and AI-powered opponent options. Designed for both casual play and competitive strategy, it showcases game development, UI design, and algorithmic decision-making.
